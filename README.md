# CRIME CITY 2D

スマホで動く、2D見下ろし型クライム・アクションゲーム（バニラJS + Canvas）。
1997年の初代GTA(2D見下ろし)の**遊びの仕組み**を再現したオリジナル作品。グラフィック・街・名称はすべて自作で、既存作品のアセット・商標は使用していません。

## 遊び方
- 移動: 左下バーチャルスティック / WASD・矢印
- 乗降: 右下 [E] / Eキー（最寄りの車をジャック）
- 射撃: 右下 [F] / Space・J
- ハンドブレーキ: 右下 [HB] / Shift

手配度★を上げると 市警→SWAT→連邦→軍 が出動。整備工場で手配度クリア、病院前でリスポーン。公衆電話からミッション開始。

## 起動（ローカル）
```bash
python3 -m http.server
# http://localhost:8000
```
